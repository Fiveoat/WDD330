    HTML5 and CSS3 for the Real World

  • Use transform in CSS to do transformations.
  • Common transformations:
  • translateX()
  • translateY()
  • scale()
  • rotate()
  • skew(x,y)
  • You can also use transform-origin.
  • Transitions
  • transition-property property defines the CSS properties of the element that should be transitioned, with all for all properties being the default
  • You can chain multiple transitions.
  • Important:
  • transition-property
  • transition-duration
  • transition-timing-function
  • transition-delay

    • Animations

  • Use @keyframes to create animations
  • Define animations that can be used.
  • animation-name: is used to attach an animation to an element
  • animation-duration: the time an animation takes to complete one iteration
  • animation-timing-function determines "how" the animation will progress over its duration
  • animation-iteration-count: number of times the animation will run
  • animation-fill-mode property defines what happens before the first animation iteration begins and after the last animation iteration concludes
  • example: .verbose { animation-name: appearDisappear; animation-duration: 300ms; animation-timing-function: ease-in; animation-iteration-count: 1; animation-direction: alternate; animation-delay: 5s; animation-fill-mode: backwards; animation-play-state: running; }

    • Canvas, SVG, and Drag and Drop

  • You can "draw" shapes and things with Canvas.
  • Some browsers don't support canvas.
  • Images can be inserted in the canvas.
  • You can also manipulate video as well.
  • SVG: Scalable Vector Graphics
  • Vector images preserve their quality even as you blow them up or shrink them down
  • You can also create shapes with SVG
  • You can apply animations to SVG
  • add the draggable attribute to make images draggable
  • You can use the dragover event handler to handle drag & drops onto.
