  • From Mobile Master UX: Lessons on UX. Do mobile devlopment first when working on projects. First create the mobile verion & then scale it up to the desktop version. Will be more clean. Creating a better user experience == $$$. No long forms. Simplify form inputs. Don't use pop-ups. Get creative & accent page with ads with content being the main attraction, not other way around.
