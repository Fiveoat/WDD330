    • Don't need to make a class in order to make a method on an object.
    • "In JavaScript, keyword this behaves unlike most other programming languages. It can be used in any
    • function, even if it’s not a method of an object."
    • "If there’s this inside a function, it expects to be called in an object context."

    • this in global context: Likely will not use.
    • this in object construction: You can use "this" when creating new objects
    • this in an object method: Use in referening the object in a method call.
    • this in a simple function: this is set to Window by default similar to global use.
    • this in an arrow function: Same as simple function except context stays in object.
