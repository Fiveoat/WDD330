    Browser Object Model

  • collection of properties and methods that contain information about the browser and computer screen
  • alert(), confirm(), and prompt() are methods of the window.
  • Get user agent window.navigator.userAgent
  • Get current page window.location.href
  • Get protocol window.location.protocol
  • Get url window.location.host
  • Get current pathname window.location.pathname
  • Get querystring params window.location.search
  • window.history.go(1); // goes forward 1 page
  • window.history.go(0); // reloads the current page
  • window.history.go(-1); // goes back 3 pages
  • new window can be opened using the window.open() method
  • Get screen dimensions window.screen.height, window.screen.width, window.screen.availWidth, window.screen.availHeight

    • Document Object Model

  • Cookies are small files that are saved locally on a user’s computer.
  • They help the client side computer "remember" user data.
  • How to save a cookie.
  • document.cookie = “name=Superman”
  • List cookies : document.cookie
  • document.cookie = “name=Batman; expires=” + expiryDate.
  • var expiryDate = new Date(); var tomorrow = expiryDate.getTime() + 1000 * 60 * 60 * 24; expiryDate.setTime(tomorrow);
  • Deleting Cookies
  • To remove a cookie, simply set it to expire at a time in the past.

    • HTML5 APIs

  • Can save data locally.
  • if(window.localStorage) { localStorage.setItem(name, “Walter White”); }
  • Get stored value.
  • if(window.localStorage) { localStorage.getItem(name); }
  • Delete from local storage.
  • if(window.localStorage) { delete localStorage.name; }
  • Clear All.
  • if(window.localStorage) { localStorage.clear(); }
  • Audio tags to add audio
  • Video tags to add video
  • Get video video = document.getElementsByTagName(“video”)[0];
  • Video methods: video.pause(); video.volume = 90; video.muted = true; video.play();
