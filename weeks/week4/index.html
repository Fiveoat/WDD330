    Javascript: Novice to Ninja Forms OOP

  • 3 main concepts: encapsulation, polymorphism, and inheritance
  • encapsulation: the inner workings are kept hidden inside the object and only the essential functionalities are exposed to the end user, such as the “on” button
  • polymorphism: the same process can be used for alternative objects
  • inheritance: taking the features of one object and then adding some new features
  • Classes not currently supported, but will be soon. Use prototypes
  • Can use constructors
  • ex : var Turtle = function(name) { this.name = name; this.sayHi = function() {return “Hi dude, my name is ” + this.name; }}
  • Add properties: Turtle.prototype.weapon = “Hands”;
  • & methods: Turtle.prototype.attack = function(){return this.name + ” hits you with his ” + this.weapon;
  • Dot notation access of properties / methods
  • Create an instance of an object & initialize : batman = Object.create(Superhuman); & batman.init(“Batman”,“Bruce Wayne”);
  • You can use monkey-patching to add methods to JS built-ins
